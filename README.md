## Project Title
p53 Structure & DNA Binding Analysis using Python

## Overview
This project analyzes the 3D structure of the p53 protein and identifies DNA-binding residues using Python. The goal is to understand how protein structure relates to DNA interaction and how mutations may affect p53 function.

## Research Question
How does the structure of p53 influence its ability to bind DNA, and how can mutations disrupt this function?

## Dataset
- Protein Data Bank (PDB)
- Structure used: 1TUP (p53-DNA complex)
- Mutant structures were also analyzed for comparison

## Tools & Libraries
- Python
- Biopython
- py3Dmol
- NumPy
- pandas
- matplotlib
